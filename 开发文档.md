# 金华聚火表格处理 - 开发文档

## 一、项目概述

### 1.1 项目简介
金华聚火表格处理是一个 Windows 桌面应用程序，用于批量处理 Excel 表格数据，将销售订单和客户明细自动转换为优路达导入模板格式。

### 1.2 技术栈
| 层级 | 技术 | 版本 |
|------|------|------|
| 语言 | Python | 3.13+ |
| GUI | tkinter (内置) | - |
| 表格读取 | pandas + xlrd + openpyxl | 3.0.1 / 2.0.2 / ≥3.1.5 |
| 打包 | PyInstaller | 6.x |
| 安装包 | Inno Setup | 6.x |

### 1.3 目录结构
```
D:\WFR\D2Y\
├── 金华聚火表格处理_整合版_fixed.py   # 主程序
├── 金华聚火表格处理_AI生成提示词.md    # 需求规范文档
├── 开发文档.md                        # 本文档
├── favicon.ico                        # 程序图标

├── build_exe.bat                      # 编译批处理

```

## 三、核心功能详解

### 3.1 明细表格处理
- **输入**: `D:\明细表格\销售订单详细 - *.xls`
- **处理**: 提取商品名称/规格/单位/价格/数量/金额，生成格式化备注串
- **输出**: `D:\明细表格\MXBG2\{订单号}.xls`，T3单元格存放汇总

### 3.2 订单表格处理（8步流水线）

#### 步骤1: 销售订单读取
- 自动识别表头行（关键字段: 客户单位/地址/订单号/联系信息/备注）
- 从 `联系信息` 列提取手机号（最后一个 `-` 后的11位数字）
- 从 `联系信息` 列提取联系人姓名（最后一个 `-` 前的内容）
- 地点名称提取: `★` 后内容 > 第一个空格后内容 > 原内容

#### 步骤2: 客户经纬度匹配
- 读取 `客户经纬度.xls`，建立 `客户单位 → 地图经纬度` 映射
- 匹配成功填入 `精准坐标` 列，未匹配的收集到警告列表

#### 步骤3: 明细表格匹配
- 扫描 `MXBG2` 目录下所有 Excel 文件
- 文件名数字与订单号双向匹配（`num in order_no or order_no in num`）
- 匹配成功则读取 T3 单元格内容，插入到备注最前面

#### 步骤4: 相同客户合并
- 按 `客户名称` 分组
- 订单号: 英文逗号合并去重
- 备注: 换行合并，末尾追加 `合并订单：N`

#### 步骤5: 备注信息处理
- 从 T3 内容提取 `=` 后数字，累加计算总金额
- 备注编号: 多行时第一行 `备注：1、`，后续 `      2、`
- 重构顺序: T3内容 → 总金额 → 合并行(紧接) → 其他 → 编号备注

#### 步骤6: 添加客户信息
- 格式: `客户名称，手机号，详细地址，客户经理`
- 日期规则: 12点后使用第二天日期

#### 步骤7: 模板填充
- 读取 `优路达导入模板.xlsx`，清空数据行保留表头
- 按列索引: 客户名称/详细地址/订单号/手机号/客户经理/备注信息/地点名称/精准坐标

#### 步骤8: 保存输出
- 文件名: `优路达导入模板{月日}-{序号}.xlsx`
- 序号: 1000-9999 随机，检查同天已有文件递增

### 3.3 商品明细统计
- 读取原始明细文件，按 `商品名称-规格-单位` 归类
- 累加 `订货数量`，等宽字体对齐显示
- 不同商品名称交错背景色（4色轮换）

### 3.4 启动优化
- 懒加载 pandas（`_LazyPandas` 代理类）避免启动阻塞
- 后台线程预加载，按钮初始显示"加载中..."

### 3.5 实时日志
- `PrintToWidget` 类重定向 stdout 到结果窗
- 处理过程实时滚动显示

## 七、构建与部署

### 7.1 开发运行
```powershell
python "金华聚火表格处理_整合版_fixed.py"
```

### 7.2 打包 EXE
```powershell
python -m PyInstaller --noconsole --onedir --name JinhuaJuhuo --icon favicon.ico --add-data "favicon.ico;." "金华聚火表格处理_整合版_fixed.py"
```

### 7.3 制作安装包
1. 安装 [Inno Setup](https://jrsoftware.org/isinfo.php)
2. 右键 `setup.iss` → Compile
3. 输出: `installer\金华聚火表格处理_Setup.exe`

### 7.4 安装包特性
- 中文安装向导
- 桌面快捷方式
- 开始菜单入口
- 完整卸载支持
- 管理员权限安装

## 八、已知限制

1. 硬编码 D 盘路径，不支持自定义目录
2. 仅支持 `.xls`/`.xlsx` 格式
3. 依赖 `联系信息` 列格式 `名称-手机号`
4. Windows 专用（依赖 `os.startfile` 和 tkinter）
5. 单线程处理，大数据量时假死
